# Plan 02: Create All 6 Domain Slices

<objective>
Create the 6 Zustand slice files under `src/store/slices/`, each containing domain-specific state and actions extracted from `useStore.ts`. Each slice follows the Zustand slice pattern: `(set, get) => ({ ...state, ...actions })`.
</objective>

<must_haves>
- Each slice file under 200 lines
- All actions from useStore.ts are represented in exactly one slice
- Each slice uses `StateCreator` typing from Zustand
- Cross-slice calls use `get()` (e.g., `get().pushHistory()` in taskSlice)
- Server action imports move to their respective slice files
</must_haves>
